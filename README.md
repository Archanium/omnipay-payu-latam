# omnipay-payu-latam
Omnipay Payu-Latam integration for https://www.payulatam.com/


## Implementation

Add this repository to `composer.json`

```json
{
    "name": "mono/v5",
    "repositories": [
        // ...
        {
            "type": "vcs",
            "url": "https://github.com/monosolutions/omnipay-payu-latam"
        },
        // ...
}
```
